字符串是不可以变得

原因：

1.底层的逻辑实现
java的String类是不可以被子类继承的
内部数组被 final 和 private 修饰：
在 Java 8 及以前：private final char[] value;
在 Java 9 及以后：private final byte[] value;（为了节省空间，引入了 Compact Strings）
final 保证了数组的引用不能指向其他数组，private 保证了外部无法直接操作这个数组内部的元素，且 String 类内部没有提供任何可以修改该数组内容的方法（没有 Setter）。
