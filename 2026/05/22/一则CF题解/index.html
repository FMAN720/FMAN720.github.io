题目大意

就是给你一个数组，然后让你在这个数组里面选择子数组，然后给这些你选则的数字加上k，这个k也是你自己选的。

分析

可以这么理解，就是给那些凹下去的地方填上相同的“砖块”，让总体是一个台阶，也就是非递减的（所有数字相同也是可以的）


这个的作用显而易见就是，求出那个凹下去最凹的差距是多大，比如
10 1 8 12
这个例子，这个凹下去最大的地方就是10-1=9，然后把i下标push进down数组

如果down是空的，那么很明显就是这个数组已经没有要改的了，因为a[i]都<=a[i+1]，已经是非递减的了，直接输出yes

这一段的意思是：
假设down数组里面是这样的 【1，3，5】
也就是说下标1的值大于下标2的值，下标3的值大于下标4的值，也就是说在2~3（for(int j=down[i-1]+1;j

void solve(){
    int n;
    cin>>n;
    vector<int>a(n+1);
    for(int i=1;i<=n;i++) cin>>a[i];
    vector<int>down;
    int k=0;
    for(int i=1;i<n;i++){
        if(a[i]>a[i+1]){
            down.pb(i);
            k=max(k,a[i]-a[i+1]);
        }
    }
    if(down.empty()){
        cout<<"YES"<<endl;
        return;
    }
    for(int i=1;i<(int)down.size();i++){
        int mx=0;
        for(int j=down[i-1]+1;j<down[i];j++){
            mx=max(mx,a[j+1]-a[j]);
        }
        if(mx<k){
            cout<<"NO"<<endl;
            return;
        }
    }
    cout<<"YES"<<endl;
}
