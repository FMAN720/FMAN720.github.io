static中的要注意的地方


报错的原因是，main是static的，但是test不是静态函数，所以会报错，如果给test加上static的话就可以了
有static修饰的方法，他不可以直接调用非静态的数据成员
但是没有static修饰的方法，可以直接调用非静态或者静态的数据成员
